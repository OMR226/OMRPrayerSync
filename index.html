Your prayer schedule, reminders, and Adhan — automated on Windows.

OMR Prayer Sync is a Windows tray app paired with a background service that keeps daily prayer times accurate, plays Adhan at the official time, reminds you after your chosen delay for each salah, and gives you a clean settings experience for prayer timing, custom reminders, and Friday Surah Al-Kahf.

Prayer times

OMR Prayer Sync keeps today’s official prayer times visible inside the app and uses those times as the foundation for Adhan and reminders.

  • Supports calculation method selection
  • Supports Standard and Hanafi Asr
  • Refreshable official times grid
  • DST-aware schedule handling

Reminders

Set a default reminder offset for all prayers, then fine-tune each salah separately. The app can remind you to pray and announce how long remains before the next prayer.

  • Default offset plus per-prayer overrides
  • Voice reminder support
  • Next-prayer countdown in reminder text
  • Name-based personalization support

Adhan

Adhan can be enabled globally and controlled per prayer. Audio playback is kept separate from spoken reminders so the experience stays respectful and clean.

  • Per-prayer Adhan toggle
  • Custom audio path support
  • Separate Fajr Adhan file support
  • Speech waits until Adhan finishes

Location

Use automatic location when available, fall back to IP lookup when needed, or lock the app to manual coordinates for a stable setup.

  • GPS preferred mode
  • IP-based fallback mode
  • Manual latitude and longitude support
  • Useful for travel or desktop-only setups

Advanced settings

Control update frequency, location freshness, API endpoint, and testing tools from one place.

  • Prayer API base URL
  • Location update interval
  • Location freshness control
  • Built-in test notification

Custom reminders & Friday reminders

Beyond prayer reminders, the app includes a customizable reminder engine and a dedicated Surah Al-Kahf tab for weekly Friday use.

  • Recurring custom reminders with weekday selection
  • Popup and audio options
  • Dedicated Surah Al-Kahf schedule
  • Optional MP3 playback

FAQ

Will the app talk over Adhan?

No. Spoken reminders are queued and played only after Adhan finishes.

Can I use different reminder delays for each prayer?

Yes. Keep a default offset or override each prayer individually.

Does it handle daylight saving time correctly?

Yes. The app now uses the correct local DST-adjusted prayer schedule.

Can I disable Surah Al-Kahf reminder or audio?

Yes. Reminder and audio can both be toggled independently, and the Friday time is customizable.
